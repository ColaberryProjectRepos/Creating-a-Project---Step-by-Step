Addding Documentation Steps Overview
********************************************************************


The  1st set of steps that comes at the beginning of every Colaberry project is a Project Overivew.  This is a set of steps designed to give the user an overall synopsis of the project.  These steps will include...


  •                     Visual Problem Statement
  •                     Visual Data Dictionary
  •                     Executive Summary
  •                     Visualize Key Insights/Key Takeaways
  •                     Visual Suggesstions/Next Steps


This documentation will include a visual representation (Power Point presentation) of each step to give a high level overview of the project.  This will also be seen by employers looking at your BIO.  We will explain in detail how to create each section by proceeding to the next step.



Instructions:

  1. Assign this task to yourself.
  2. Once you have read through this task and you fully understand it, write a small paragraph with screenshots and notes of what you learned.  Then add it as a comment on this task.
  3. Everyday you work on this task, you will add a minimal of a comment and a screenshot.  Gifs are good as well.
  4. Note: You as well as other people will be reading through your project notes.  Make sure you talk about your thoughts at the moment so you and others can relive the experience.
  5. Upon completion, close the task and navigate back to your Colaberry Project Dashboard.
