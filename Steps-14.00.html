Once you have been approved for Stress Test 2, you will need to prepare your Step by Step instructions and store them in this ticket so when you get to Stress Test 4, they are easier to retrieve.


  1. Every object in your project will get a set of instructions combined with a gif following those instructions.  You have the ability to combine objects as you see fit, for example, if you have a dashboard you could combine the title and logos in the same step.
  2. Format should be ...
    1. Screenshot of object (no application in screenshot)
    2. Numbered instructions
    3. GIF  
      1. Gifs should include the application (For example PowerBI or SQL Server) in video
      2. Keep all gifs < 4MB.  If it's bigger than that, you need to break it out into two steps.
      3. Besides the project heading, each gif should only show the item you are building for that step.
      4. Follow your numbered instructions in the gif
  3. Your 1st Step in the project should be details on how to retrieve the data.  Your gif can show you going to where the data is and downloading or injesting it.  Instructions should include all details that are needed for someone else to download the same dataset.  (No basecamp links)
